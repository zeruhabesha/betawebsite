Drop your .mp4 footage here to activate real video in the site.

Expected filenames (see ../../README.md for details):
  hero-cyber-defense.mp4   -> Home hero (a defender protecting infrastructure)
  platform-network.mp4     -> Platform page
  soc-operations.mp4       -> Home / GRC pages
  threat-stream.mp4        -> About page

Until a file is present, a themed poster image is shown instead, so every
section still looks complete. Recommended: 1920x1080, H.264, muted, ~10-20s
loop, under ~6 MB. Free sources: Pexels Videos, Mixkit, Coverr.
